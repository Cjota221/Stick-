# Arquivos da marca

Coloque os arquivos nesta pasta com estes nomes:

- `logo.png` - logo principal, preferencialmente com fundo transparente.
- `favicon.png` - ícone quadrado, preferencialmente com 512 x 512 px.

Também podem ser enviados arquivos SVG ou ICO. Depois do upload, os nomes podem
ser ajustados no projeto conforme os arquivos recebidos.
